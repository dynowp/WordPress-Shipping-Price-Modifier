Modify your WooCommerce shipping rates according to your needs.

== Description ==

Freight Modifier is a WooCommerce plugin that allows WordPress administrators to modify shipping rates based on specific settings. With this plugin, you can either increase or decrease the shipping cost, either by a fixed amount or a percentage.

== Features ==

- Enable or disable the modification of shipping rates.
- Choose between increasing or decreasing the shipping cost.
- Set the modification amount as a fixed value or a percentage.
- Activity log option, allowing you to monitor changes applied to shipping rates.
- Simple and direct integration with WooCommerce.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Access the plugin settings in WooCommerce > Settings > Freight Modifier to adjust the options.

== Frequently Asked Questions ==

= Does the plugin support negative shipping values? =

Yes, you can configure it to apply a discount on the shipping rate, resulting in a reduced final value.

= Is the plugin compatible with all versions of WooCommerce? =

The plugin has been tested with WooCommerce version 8.4.0. We always recommend using the latest version of WooCommerce to ensure the best compatibility.
